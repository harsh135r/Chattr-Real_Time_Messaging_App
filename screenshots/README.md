## ✨ Features

- 🔴 Real-time messaging powered by live data sync
- 👤 User authentication (via phone number)
- 💬 One-on-one chat conversations
- 🔍 User search through username
- 🔔 Push notifications for new messages
- 🕐 Message timestamps

## 📋 Requirements

- Android **10.0 (API Level 29)** or higher
- Android Studio **Hedgehog (2023.1.1)** or later
- JDK 11+
- A Firebase project (for backend services)

---

## 📱 Usage

1. **Sign Up / Log In** — Create an account and set a username or log in with your phone number
2. **Find a User** — Search for other registered users to start a conversation
3. **Send Messages** — Type and send messages in real time
4. **Receive Notifications** — Get notified when someone messages you, even when the app is in the background
